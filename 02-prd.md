# 소코반(Sokoban) 게임 PRD

## 1. 개요

상자를 밀어 목표 지점에 모두 옮기면 클리어되는 클래식 퍼즐 게임 **소코반**을 제작한다.
복잡한 학습 없이 누구나 바로 즐길 수 있는 **가볍고 단순한 퍼즐**을 지향한다.

| 항목 | 내용 |
| --- | --- |
| 게임 장르 | 퍼즐 (밀기 퍼즐) |
| 플랫폼 | 웹 / 모바일 (짧은 세션 플레이) |
| 핵심 가치 | 단순함, 빠른 시작, 짬짬이 즐기기 |

---

## 2. 타깃 사용자

- **단순한 퍼즐 놀이를 좋아하는 사용자**
  - 복잡한 규칙이나 튜토리얼 없이 바로 이해할 수 있는 게임을 선호
  - 가벼운 두뇌 자극을 원하는 캐주얼 플레이어

---

## 3. 사용 목적 (Why)

- **버스, 지하철에서 심심할 때** 짧게 즐기기 위함
- 이동 중 한 손/짧은 시간으로도 부담 없이 플레이 가능해야 함
- 한 판이 길지 않고, 중간에 끊겨도 부담 없어야 함

> 설계 시사점: 짧은 세션, 빠른 로딩, 간단한 조작, 즉시 재시작이 중요하다.

---

## 4. 핵심 기능 (What)

### 4.1 단계 (Stage)

- **1~10단계**로 구성한다.
- 단계가 올라갈수록 난이도가 점진적으로 상승한다.
  - 초반(1~3단계): 상자 1~2개, 조작 익히기
  - 중반(4~7단계): 상자 2~3개, 동선 고민 필요
  - 후반(8~10단계): 상자 3개 이상, 막힘 주의 필요
- 한 단계를 클리어하면 다음 단계로 이동한다.
- 마지막 10단계 클리어 시 **전체 클리어** 화면을 보여준다.

### 4.2 배경 (Background)

- **벽돌집** 테마를 기본 배경으로 사용한다.
  - 벽: 벽돌 텍스처
  - 바닥: 벽돌집 내부 바닥 느낌
  - 통일감 있는 따뜻한 톤으로 편안한 분위기 연출

### 4.3 애니메이션 (Animation)

- 캐릭터와 상자가 **움직일 때 부드러운 애니메이션**을 적용한다.
  - 칸 단위 이동을 딱딱하게 점프시키지 않고 **부드럽게 슬라이드**
  - 이동 전환에 짧은 보간(easing) 적용 (예: ease-in-out, 약 100~150ms)
  - 빠른 연속 입력에도 끊김 없이 자연스럽게 이어지도록 처리

---

## 5. 게임 규칙

- 플레이어는 상하좌우 한 칸씩 이동한다.
- 플레이어 진행 방향에 **상자**가 있으면 밀 수 있다.
  - 단, 상자 뒤가 벽이거나 다른 상자면 밀 수 없다.
  - 상자는 당길 수 없다 (밀기만 가능).
- **모든 상자를 목표 지점(target)에 올려놓으면 클리어**된다.
- 막혀서 더 이상 풀 수 없을 때를 대비해 **되돌리기(Undo)** 또는 **재시작**을 제공한다.

### 구성 요소

| 요소 | 설명 |
| --- | --- |
| 플레이어 | 사용자가 조작하는 캐릭터 |
| 상자 | 밀어서 옮기는 대상 |
| 목표 지점 | 상자를 올려야 하는 칸 |
| 벽 | 통과/이동 불가 |
| 바닥 | 이동 가능한 칸 |

---

## 6. 조작 방식

- **키보드**: 방향키 (↑ ↓ ← →) 또는 WASD
- **터치/모바일**: 화면 스와이프 또는 방향 버튼
- 보조 조작:
  - 되돌리기(Undo)
  - 단계 재시작(Restart)

---

## 7. 화면 구성

1. **시작 화면**: 게임 시작 / 단계 선택
2. **게임 화면**:
   - 벽돌집 배경 위의 퍼즐 맵
   - 현재 단계 표시
   - 되돌리기 / 재시작 버튼
3. **클리어 화면**:
   - 단계 클리어 시 → 다음 단계로 이동
   - 10단계 클리어 시 → 전체 클리어 축하 화면

---

## 8. 비기능 요구사항

- **빠른 로딩**: 이동 중에도 즉시 실행 가능
- **가벼움**: 단순한 그래픽과 적은 리소스
- **반응성**: 입력에 즉각 반응, 애니메이션이 조작감을 해치지 않음
- **간결한 UI**: 핵심 정보(단계, 조작 버튼)만 노출

---

## 9. 향후 확장 (선택)

- 단계 추가 (11단계 이상)
- 이동 횟수 / 최소 이동 기록
- 다른 배경 테마
- 효과음 / 배경음
